# Pasos Importantes para vincular un archivo HTML con uno CSS

1. <link rel="stylesheet" href="estilos.css"> primero usamos link rel="stylesheet", despues: href="titulo.css"
2. Para llamar a clases en css: utilizamos primero punto(.) ej: .text_blue{}
3. Para llamar a id (cuando UN elemento necesita un UNICO estilo), utiliza # ej: #titulo_principal{}

# Propiedades comunes en CSS
- color; rgb(255,255,255): el color del texto
- background color: yellow : Elemento del background (fondo)
- font-size: 10px: tamaño del texto
- margin: 10px: espacio de afuera
- padding: 10px: espacio de adentro
- border: 1px solid black: el borde
- width/height: 200px: el tamaño
- text-align: center: alineación del texto
